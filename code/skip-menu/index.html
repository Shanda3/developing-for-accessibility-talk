Tips: Always check the tab-ability of your site. Tab follows the order of your HTML, so if you use CSS styles like reverse flex-direction or float to modify content order, then that might affect your tab order.

Different sites need different levels of skip menus; but at a minimum you should include one that allows bypassing your main navigation.
