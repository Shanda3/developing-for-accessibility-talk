Images & Figures

Background Image

Best for:

  • website aesthetics
  • no significant content

Image Tag

Best for:

  • significant content
  • no label needed/desired

Figure + Caption

Best for:

  • significant content
  • label needed/desired

Tips:

  • Use "alt" text for your content's images, which you can't set on background images.
  • Image alt text should be functional, while figcaption text should be editorial and descriptive.
  • Only use background images for background content. They may often be easier to style size-wise, but they should be considered ignorable by users as far as content goes. You also can't "save" them from the browser.
  • For even more flexibility, see the picture HTML element. Note: not supported on IE or Opera Mini.
