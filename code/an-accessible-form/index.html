Tips:

  • Placeholder text isn't a label. You need an actual label that is tied to the field it is labelling.
  • Make sure your tab order flows properly.
  • Make sure you have good focus indication.
  • Group related fields (like radio buttons and checkboxes) together with fieldsets and legends
  • Use ARIA properties if you are doing anything beyond what semantic HTML supports.
